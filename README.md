# 鬼滅之刃劇照圖庫

這是一個簡單的網頁應用，提供鬼滅之刃動畫劇照的展示與下載功能。

## 功能特點

- 展示鬼滅之刃劇照圖片
- 提供一鍵下載功能
- 響應式設計，適合各種裝置瀏覽

## 技術實現

- 純 HTML、CSS 和 JavaScript 實現
- 使用 ES6 語法
- 無需額外依賴庫

## 使用方法

1. 直接開啟 `index.html` 檔案即可在瀏覽器中瀏覽
2. 點擊圖片下方的下載按鈕可下載對應劇照

## 檔案結構

- `index.html` - 主要網頁文件
- `鬼滅之刃無限城編.png` - 無限城編劇照
- `鬼滅之刃》劇場版「無限城篇」.jpg` - 劇場版劇照
